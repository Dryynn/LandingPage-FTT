# FTT - Landing Page

## Sobre o Projeto
Esta landing page foi desenvolvida para a **FTT (Fábrica de Tecnologias Turing)**. O projeto foi construído com base em um design elaborado por Product Designers (PDs), focando em uma estética moderna e profissional para conectar o ambiente acadêmico ao mercado. Vale ressaltar que, embora o desenvolvimento tenha sido concluído, o design final não foi aprovado para uso oficial.

Acesse aqui: [FTT - Landing Page](https://landing-page-ftt.vercel.app/)

## Tecnologias
O projeto utiliza as seguintes tecnologias e bibliotecas:
- **React 19**: Biblioteca principal para construção da interface.
- **Vite**: Ferramenta de build e servidor de desenvolvimento ultra-rápido.
- **Lucide React**: Biblioteca de ícones modernos e leves.
- **CSS Vanilla**: Estilização personalizada focada em performance e flexibilidade.
- **SVG Dinâmico**: Backgrounds topográficos e overlays de grade criados via código.

## Funcionalidades
- **Design Premium**: Interface em modo escuro com estética moderna, uso de gradientes e efeitos de profundidade.
- **Hero Interativo**: Seção inicial com animações de "code-drop" e camadas isométricas.
- **Seções de Conteúdo**:
  - **Sobre**: Detalhamento da missão da FTT.
  - **Motivos**: Vantagens para empresas e estudantes.
  - **Processo**: Fluxo de funcionamento dos projetos.
  - **Impacto**: Resultados e importância da iniciativa.
- **CTA (Call to Action)**: Botões de ação estratégica para diferentes perfis de usuário.
- **Totalmente Responsivo**: Adaptado para proporcionar uma excelente experiência em dispositivos móveis e desktops.

## Objetivos
- **Conectar Talentos**: Atuar como uma plataforma que une estudantes talentosos a empresas e organizações reais.
- **Experiência Prática**: Proporcionar aos estudantes a oportunidade de desenvolver projetos reais e ganhar experiência de mercado.
- **Inovação**: Levar a inovação e o conhecimento acadêmico diretamente para o setor produtivo.

## Como Executar

Para rodar este projeto localmente, siga os passos abaixo:

1. **Clonar o repositório:**
   ```bash
   git clone [url-do-repositorio]
   ```

2. **Acessar a pasta do projeto:**
   ```bash
   cd FTT
   ```

3. **Instalar as dependências:**
   ```bash
   npm install
   ```

4. **Iniciar o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Acessar no navegador:**
   O terminal exibirá um endereço (ex: `http://localhost:`). Basta abrir este link para visualizar a landing page.
